🔧 Tech Stack components

📱 Android Architecture

  • Activities: 15+ screens serving modular functionalities.
  • Services: Persistent Voice, GPS, and BLE listeners.
  • Receivers: Bound to system events (SMS readiness, boot up).
  • WorkManager: Used for periodic syncing tasks.
Kotlin WorkManager

🧩 Third-Party Integrations

  • Supabase: For Database, Auth, Realtime APIs, and Media Storage.
  • Firebase: Dedicated to FCM Push Notification pipelines.
  • Vosk: Offline embedded Speech Recognition (Wake Word).
  • Google Maps: Native map SDK rendering.
Vosk AI Supabase Firebase

✨ Core Workflows

  • Identity: Splash, Auth Session, Profile Creation.
  • Monitoring: Home Dashboard, Status Flags, SOS.
  • Social/Network: Add Guardian, Nearby Helpers.

User Hardware Benchmarks

  • OS Version: Android 8.0 (API 26) or newer.
  • Memory/Storage: Min 3GB RAM (4GB desired) / 500MB+ Application space + Cache.
  • Required Nodes: Dedicated GPS Hardware, active Microphone, BLE Chipset, Camera.
